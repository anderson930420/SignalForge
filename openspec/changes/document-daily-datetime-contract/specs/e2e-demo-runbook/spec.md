## MODIFIED Requirements

### Requirement: Demo runbook documents validation boundaries

The SignalForge-to-AlphaForge E2E demo runbook SHALL document the file-based handoff, repository metadata, and validation boundaries.

#### Scenario: Demo documents daily alignment boundary

- **WHEN** a user reads the E2E demo runbook
- **THEN** it states the demo uses daily trading-date alignment
- **AND** it states the demo does not test intraday timing semantics
